## Usage

See the [documentation](https://mhgbrown.github.io/vue-cheeseburger-button/dist/).

## Build Setup

``` bash
# install dependencies
npm install

# serve doc app with hot reload at localhost:8080
npm run start

# build for production with minification
npm run build

# run unit tests
npm run unit
```
